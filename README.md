# 📊 HR Analytics Dashboard | Power BI Project

## 📌 Project Overview

The **HR Analytics Dashboard** is an interactive Power BI project designed to analyze employee data and uncover workforce trends, employee attrition patterns, and key HR metrics. By transforming raw HR data into meaningful visualizations, this dashboard helps HR professionals make informed, data-driven decisions to improve employee retention and workforce planning.

The project includes:

- ✅ Data Cleaning & Transformation
- ✅ HR Data Analysis
- ✅ KPI Creation using DAX
- ✅ Interactive Power BI Dashboard
- ✅ Employee Attrition Analysis

---

# 🎯 Project Objectives

- ✔️ Analyze employee attrition rate
- ✔️ Identify factors influencing employee turnover
- ✔️ Study attrition based on age, salary, education, job role, and years at the company
- ✔️ Build an interactive HR analytics dashboard
- ✔️ Generate actionable insights for HR teams
- ✔️ Support data-driven workforce planning

# 🔍 Key Insights

The analysis revealed several important HR insights:

- 📌 The highest employee attrition occurred in the **26–35 age group**.
- 📌 Employees in lower salary ranges experienced higher attrition.
- 📌 Employees with **less than two years** at the company were more likely to leave.
- 📌 Certain job roles and education fields showed significantly higher turnover.
- 📌 Job satisfaction strongly influenced employee retention.
